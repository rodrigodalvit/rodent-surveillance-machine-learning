# Rodent Tracking-Plate Classification Using Image Processing and Machine Learning

This repository provides the MATLAB code used for the image-processing and machine-learning workflow associated with the study:

**Incorporating Machine Learning Techniques to Enhance Rodent Surveillance in Marginalized Urban Communities**  
*Ecology and Evolution*  
DOI: `10.1002/ece3.72382`

The project applies computer vision and machine-learning methods to classify rodent tracking-plate images. The workflow was developed to support faster, more reproducible interpretation of tracking plates used in urban rodent surveillance.
